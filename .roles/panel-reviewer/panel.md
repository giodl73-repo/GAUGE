# Rail Peer Panel (illustrative)

Archetype peer reviewers for GAUGE research outputs — distinct from the parliament
(adversarial design review) and the editorial gate (form review). Each is an
archetype, not a named person.

| ID | Reviewer archetype | Expertise |
|---|---|---|
| R-P1 | Transport planner | Intercity/HSR network planning, ridership forecasting |
| R-P2 | Network scientist | Graph algorithms, connectivity, centrality on spatial networks |
| R-P3 | Transport economist | Benefit-cost, demand modeling, modal diversion |
| R-P4 | Rail operations researcher | Timetabling, OTP, shared-track dispatch, capacity |
| R-P5 | Climate / mobility researcher | Modal shift, lifecycle carbon, electrification |

Assignment guide:

- Topology / connectivity method → R-P2, R-P4
- Ridership / benefit-cost → R-P1, R-P3
- Modal shift / climate → R-P5, R-P3
- Tier/SLA + reliability methodology → R-P4, R-P1
